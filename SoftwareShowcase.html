Sticknodes

Also called 2D animation,the typical kind of animation,you draw each pose frame by frame to create movement, movement such as walking, running and many more. In 2D animation we use simple shapes to create "flat" or 2D characters.

Stick Nodes is an animation software that is similar to Pivot, it is designed for mobile devices
